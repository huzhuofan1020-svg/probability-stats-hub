Chapter 03

多维随机变量及其分布

本章我们将随机变量的概念从一维推广到多维，重点讨论二维随机向量的联合分布、边缘分布和条件分布，以及它们之间的独立性关系。

§3.1

随机向量及其联合分布

定义 — 随机向量与联合分布函数

设 $(X_1, \cdots, X_n)$ 是定义在同一概率空间上的 $n$ 个随机变量，则称其为 $n$ 维随机向量。

联合分布函数定义为：

$$F(x, y) = P(X \le x, Y \le y)$$

性质：

  • $0 \le F(x, y) \le 1$
  • 单调不减，右连续
  • $F(-\infty, y) = F(x, -\infty) = 0, \quad F(+\infty, +\infty) = 1$
边缘分布函数

$$F_X(x) = P(X \le x) = F(x, +\infty) = \lim_{y \to +\infty} F(x, y)$$

$$F_Y(y) = P(Y \le y) = F(+\infty, y) = \lim_{x \to +\infty} F(x, y)$$

§3.2

离散型随机向量

联合分布律

$$p_{ij} = P(X = x_i, Y = y_j), \quad i, j = 1, 2, \cdots$$

独立性充要条件：

$P(X=x_i, Y=y_j) = P(X=x_i) \cdot P(Y=y_j)$ 对所有 $i, j$ 成立。

§3.3

连续型随机向量及其联合密度

联合概率密度函数

若存在非负可积函数 $f(x, y)$，使得

$$F(x, y) = \int_{-\infty}^{x} \int_{-\infty}^{y} f(u, v) \, du \, dv$$

独立性充要条件：$f(x, y) = f_X(x) \cdot f_Y(y)$ 几乎处处成立。

§3.4

随机向量函数的分布

卷积公式 (Z = X + Y)

若 $X, Y$ 独立，则 $Z = X + Y$ 的密度函数为：

$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) \, dx$$
